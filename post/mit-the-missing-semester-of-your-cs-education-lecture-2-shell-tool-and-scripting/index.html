    双引号中的变量会被展开

$ echo '$foo world'
$foo world

$ source a.sh

  • mkdir -p ( create a directory under the current directory)

  • 保留变量
    $0: the name of the script
    1−9:thesecondthroughtthenintharguments1-9: the second throught the ninth arguments 1−9:thesecondthroughtthenintharguments?: get the value from the previous command

$ mkdir test

  • grep ( search)
$ grep 1 a.cpp ( search a in the a.cpp)

  • 返回值
    0 : true
    1 : false

  • 运算符
    &&, ||, ;

$ false || echo "oops fail"
oops fail
$ true || echo "will not be printed"
$ true && echo "it goes well"
it goes well
$ false && echo "will not be printed"
$ false ; echo "things always be printed"
things always be printed
  • 命令替换: $( )
$ foo=$(pwd)
$ echo "We are in the $(pwd)"
  • 过程替换: <() 将()内容输入文件
$ echo <(ls) <(ls ..)
