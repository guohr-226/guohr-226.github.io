shell

what is shell

命令行工具，基于文本工具
windows: powershell
linux: bash ( bourne again shell）

bash

命令行提示符 ( shell prompt)

  • date ( date and time)

带参数 ( arguments) 执行程序

  • echo ( print the arguments you give it)
$ echo hello
hello

参数需要空格 ( white space)分开
字符串用"" or ''
单转义用\

$ echo "hello world"
hello world
$ echo hello\ world
hello world
  • mkdir ( make directory)

内嵌程序位于系统文件 ( system file)
搜索借助环境变量 ( environment variable)中 PATH变量 ( PATH variable)

$ echo $PATH
  • which ( to find which it actully runs)
$ which echo

PATH:
linux/macos: /
under root/
windows:
one root for every partition
absolute path/ relative path

  • pwd ( print working directory)
$ pwd
  • cd ( change directory)
$ cd /home
  • . ( the current directory)
  • .. ( the parent directory)

write a program please give a absolute path😀

运行程序默认当前目录

  • ls ( list the files in the current directory)
$ ls
  • ~ ( direct to the home directory)
  • cd - ( cd the dirctory you were previously in)

程序有标志 ( flag)和选项 ( option)作为参数，以-开头

  • --help
$ ls --help
  • -l ( give you more information)
$ ls -l

ls -l的列表：
3个字母一组
当前用户权限 | 当前用户组权限 | 其他人权限
read | write | execute
for flie: read ( read it), write ( save, add more, replace), execute ( run it)
for directory: read( list for the directory), write ( rename, create, remove file), execute( enter the dierctory)
(detele flie need the write for the directory)

  • mv ( rename file) ( change or move)
$ mv a.cpp b.cpp
$ mv a.cpp /b.cpp
  • cp ( copy)
$ cp a.cpp /b.cpp
  • rm ( remove) ( can not remove directory)
$ rm a.cpp
$ rm -r /dir ( to remove all files under the directory)
  • rmdir ( remove empty directory)
$ rmdir /dir
  • mkdir ( create a directory)
$ mkdir /a

  • crtl+c ( cancel)

  • man ( manul pages) (按q退出)

$ man ls
  • ctrl+l ( 清空终端)

流: < in > out >> add

$ echo hello > hello.out
  • cat ( show the file)
$ cat hello.in
$ cat < hello.in > hello.out
$ cat hello.out
$ cat <hello,in >> hello.out
$ cat hello.out
  • pipe:| ( take the output of the left program to and make the input of the right program)
$ a | b
  • tail ( print the last line)
$ tail -n1 ( print the last one line)

$ ls -l / | tail -n1> ls.txt
  • sudo ( super user: root)
  • sysfs ()
$ cd /sys

$ sudo su ( use root to run shell)
# ...
  • tee ( write to the file and shell)
$ tee < hello > hello.out

$ echo 500 | sudo tee brightness

  • xdg-open ( use right sofrware to open a file)

  • touch ( change the time of the file) ( usually use it to create a new file)

$ touch a.cpp
  • chmod ( change the permission)
u - user
g - group
o - other
+/-
rwx

$ chmod ug+rw a.cpp
